# 🏠 House Price Prediction

## 📌 Overview
A Machine Learning project that predicts house prices using Linear Regression. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

---

## 🚀 Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Encoding
- Linear Regression Model
- Model Evaluation
- House Price Prediction

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
